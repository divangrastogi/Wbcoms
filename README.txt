== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `subs-box.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== How to use? ==

1. Go to the woocommerce product section.
2. Select the subcription Box option from the product type list.
3. You can give the various option in "Frequency Options" to create subscription frequency ex: Monthly, Quarterly, Annually and so on.
4. You can assign the prices in "Frequency Price Options" to thier frequencies respectively using the comma (,) seperation like 24,25,26.
5. You can add the quantity for the product in the "Number of Items in Box" input box.
6. User can select the subsciption frequency and price with quantity from the single product page.
